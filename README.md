# PPT点名插件

这是一个基于 PowerPoint VSTO 的随机名字选择器插件，允许用户导入一个名字列表，并在满足条件的随机时间内随机选择一个名字。

## 功能

1. **导入名单**：用户可以从本地选择一个包含名字的文本文件进行导入。
2. **开始随机点名**：在新的窗口中滚动显示被随机打乱的名单中的名字，并在指定的随机时间段内停止定格。

## 公平性说明

该随机名字选择器插件是为了确保每个名字都有平等的被选中的机会而精心设计的：

1. **名单打乱**：在每次点击“开始随机点名”按钮时，插件首先会获取已导入的名单，并使用 Fisher-Yates 洗牌算法来随机化名字的顺序。该算法是一个经过证明的公平的打乱方法，确保每个名字都有平等的机会被放在任何位置。
  
2. **滚动显示**：随机化后的名单将被用于滚动显示。这确保了每个名字，无论其在原始列表中的位置如何，都有同等的机会被看到。

3. **随机选定**：滚动显示的持续时间是随机的，当到达这个随机时间后，滚动会停止。此时屏幕上显示的名字就是被选中的名字。由于名单已经被随机化，并且滚动时间也是随机的，这确保了每个名字都有平等的机会被选中作为最终结果。

4. **短时间滚动选择的公平性**：尽管有时滚动时间可能非常短，可能给人一种只有名单开始部分的名字才会被选中的印象，但实际上由于名单在每次滚动前都被重新打乱，每个名字都有同样的机会出现在名单的开始部分。因此，无论滚动时间长短，每个名字都有平等的机会被选中。


## 安装指南

1. 下载项目的发布包并解压。[点击这里下载最新版本](https://github.com/aiyamia/RandomNamePickerForPPT/releases/download/v0.1.0/RandomNamePicker.zip)
2. 运行安装程序setup.exe并按照提示进行安装。
3. 安装完毕后，打开 PPT，你应该可以在功能区中看到一个名为“点名”的选项卡。

## 使用说明
在“点名”选项卡中：
1. 首先，点击“导入名单”按钮，从本地选择一个包含班级所有同学名字的文本文件进行导入。请确保文本文件的每一行为一个名字。
2. 接下来，点击“开始点名”按钮，将在屏幕中央弹出一个小窗口，名字将开始在其中滚动显示。
3. 在1到2秒内，滚动将停止，选中一个随机的名字。

## 如何卸载
1. 通过Windows的“添加或删除程序”功能：
 - 打开Windows的“控制面板”。
 - 点击“程序”或“程序和功能”。
 - 在列表中找到并点击“RandomNamePicker”或与此相关的条目。
 - 点击“卸载”并按照提示完成卸载过程。

2. 如果是作为Office插件：
 - 打开PPT。
 - 点击“文件”>“选项”>“加载项”。
 - 在加载项类型列表框中，选择“COM 加载项”，然后点击“转到...”。
 - 在对话框中，找到“RandomNamePicker”，并取消选中它。
 - 点击“删除”或“卸载”按钮。
 - 点击“确定”完成卸载过程。
